<p align="center"><b>How to install on ANKI?</b></p>
<p align="center"><b>Step 1</b><br>On Anki, add a new card with Basic Note type, and go to <em>Cards</em></p>
<p align="center"><img src="https://i.imgur.com/59gU6Pp.png"></p>

<p align="center"><b>Step 2</b><br>At Cards, find the respective files on github above and copy the code into ANKI <em>Cards</em></p>
<p align="center"><img width="700px" src="https://i.imgur.com/oGPRRBA.png"></p>

<p align="center"><b>How to install on ANKI?</b><br>Same as above but before copying and pasting code, <b>first edit the <em>Fields</em></b>.</p>
<p align="center"><img src="https://i.imgur.com/ZvXPH3L.png"></p>
  
<p align="center"><b>Step 1</b><br>Add a new card with <b>Basic (and Reversed Card)</b> Note type and FIRST add a <em>Hint</em> field</p>
<p align="center" display="inline"><img width="700px" src="https://i.imgur.com/B8AniUo.png"></p>
  
<p align="center"><b>Step 2</b><br>Go to <em>Cards</em>, and copy the respective files from github and paste in to the card templates</p>
<p align="center"><b>Step 3</b><br>At <em>Cards</em>, you'll have to change to the reverse card to add the additional HTML files for the Front and Back of the flipped card</p>
